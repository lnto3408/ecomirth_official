# Privacy Policy

**card-news-sns-analyzer**

Last updated: 2026-03-15

## 1. Information We Collect

The App accesses the following data through official platform APIs:

- **Account statistics**: Follower count, following count, profile information
- **Content metrics**: Views, likes, comments, shares, saves, reach for your posts
- **Post metadata**: Captions, hashtags, posting timestamps

## 2. How We Use Your Information

All collected data is used solely to display analytics and insights within the App on your local device. Data is:

- Stored locally in a SQLite database on your computer
- Never transmitted to external servers (other than official platform APIs)
- Never shared with third parties
- Never used for advertising or profiling

## 3. Data Storage

All data is stored locally on your device at the application's data directory. You can delete all stored data at any time by removing the application data folder.

## 4. Third-Party Services

The App connects to the following APIs using your own credentials:

- TikTok API (via TikTok for Developers)
- Instagram Graph API (via Meta for Developers)
- Threads API (via Meta for Developers)

Each platform's own privacy policy applies to data accessed through their APIs.

## 5. Data Retention

Data is retained locally until you choose to delete it. Uninstalling the App removes all stored data.

## 6. Contact

For privacy-related questions, contact via GitHub Issues on this repository.
